# GitHub'a Dosya Yükleme Rehberi

## Adım 1: Ana Dosyalar
GitHub repository sayfasında "Add file" → "Create new file" ile şu dosyaları oluştur:

### pubspec.yaml
- Dosya yolu: `pubspec.yaml`
- İçeriği: pubspec.yaml dosyasının içeriğini kopyala-yapıştır

### README.md
- Dosya yolu: `README.md`
- İçeriği: README.md dosyasının içeriğini kopyala-yapıştır

### .gitignore
- Dosya yolu: `.gitignore`
- İçeriği: .gitignore dosyasının içeriğini kopyala-yapıştır

## Önemli Not:
- Her dosyayı oluştururken, dosya yolunu doğru yaz (örn: `lib/main.dart`)
- GitHub otomatik olarak klasörleri oluşturur
- Her dosyayı ekledikten sonra "Commit changes" butonuna tıkla
- Commit mesajı: "First commit: Project structure and basic screens"

## Daha Kolay Yöntem:
Tüm dosyaları tek seferde yüklemek için "Add file" → "Upload files" kullanabilirsin, ama o zaman dosyaları sürükle-bırak yapman gerekir.
